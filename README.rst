=========
ewoc-prod
=========

ewoc-prod package is in charge to run production from end to end


Description
===========

Prerequesites:
- a ewoc database
- a ewoc production cluster

This code is plan to run by default on the master node of the cluster

Aux data: AEZ with tiles id by AEZ region
